| Stage | User Action | System Response | Emotion | Screen |
|-------|-------------|-----------------|---------|--------|
| **1. Land** | Arrives at registration page | Show value proposition | Curious | Landing/Register |
| **2. Register** | Enters email, name, org name | Validate input, check duplicates | Hopeful | Registration Form |
| **3. Verify** | Clicks email verification link | Verify token, activate account | Relieved | Email Verification |

| **5. Plan** | Selects subscription plan | Show features, pricing breakdown | Evaluating | Plan Selection |
| **6. Pay** | Enters payment details | Process payment, create tenant | Committed | Payment |
| **7. Welcome** | Views success confirmation | Show dashboard, start onboarding | Excited | Welcome Screen |
| **8. Tour** | Completes guided tour | Highlight key features | Empowered | Onboarding Tour |
