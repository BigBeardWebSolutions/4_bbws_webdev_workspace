# Monitoring Agent

Manages infrastructure and tenant monitoring for the BBWS Multi-Tenant WordPress Platform.

## Responsibilities

- Infrastructure health checks
- Tenant availability monitoring
- CloudWatch alarm management
- Dashboard creation
- Incident response automation
- SLA reporting

## Files

| File | Purpose |
|------|---------|
| `agent.md` | Agent definition and workflows |
| `agent_spec.md` | Detailed technical specification |
| `skills/` | Agent-specific skill definitions |

## Related Agents

- Backup Manager - Backup health monitoring
- DR Manager - DR status monitoring
- Cost Manager - Cost-related alerts
