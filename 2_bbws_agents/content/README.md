# Content Manager Agent

Manages WordPress content operations for the BBWS Multi-Tenant WordPress Platform.

## Responsibilities

- WordPress plugin management
- Theme configuration
- Site import/export
- Content backup
- WP-CLI operations
- Troubleshooting

## Files

| File | Purpose |
|------|---------|
| `agent.md` | Agent definition and workflows |
| `agent_spec.md` | Detailed technical specification |
| `skills/` | Agent-specific skill definitions |

## Related Agents

- Tenant Manager - Provides tenant infrastructure
- Backup Manager - Content backup operations
