# Operations Runbooks

This folder contains step-by-step operational procedures for tenant deployment and management.

## Available Runbooks

### Deployment
- **SIT_TENANT_DEPLOYMENT_RUNBOOK.md** - Detailed SIT deployment procedures
- **PRODUCTION_DEPLOYMENT_RUNBOOK.md** - PROD deployment with approval gates

### Troubleshooting
- **DEPLOYMENT_TROUBLESHOOTING.md** - Common deployment issues and fixes
- **ROLLBACK_PROCEDURES.md** - How to rollback failed deployments

### Maintenance
- **TENANT_UPGRADE_RUNBOOK.md** - Upgrading tenant WordPress versions
- **DATABASE_MAINTENANCE.md** - Database backup and restore procedures

## Runbook Format

Each runbook follows this structure:
1. **Prerequisites** - What you need before starting
2. **Step-by-Step Procedure** - Detailed execution steps
3. **Validation** - How to verify success
4. **Troubleshooting** - Common issues and fixes
5. **Rollback** - How to undo changes if needed
