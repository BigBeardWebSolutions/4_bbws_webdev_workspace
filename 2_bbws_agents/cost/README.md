# Cost Manager Agent

Manages cost tracking and optimization for the BBWS Multi-Tenant WordPress Platform.

## Responsibilities

- Cost reporting and analysis
- Budget management
- Cost forecasting
- Optimization recommendations
- Tenant cost allocation
- Chargeback calculations
