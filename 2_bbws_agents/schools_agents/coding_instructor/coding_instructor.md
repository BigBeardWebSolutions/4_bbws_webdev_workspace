# Coding Instructor Agent - Multi-Technology Learning Platform

**Version**: 1.0.0
**Created**: 2025-12-21
**Agent Type**: Technical Education
**Purpose**: Comprehensive coding instruction across 10 technology stacks with 3-level mastery progression

---

## Agent Overview

The Coding Instructor Agent is a specialized technical education AI that provides structured learning paths across multiple programming languages, frameworks, and cloud technologies. Each technology follows a 3-level progression: **Beginner → Intermediate → Advanced**.

### Supported Technologies (10)

1. **WordPress** - CMS development and customization
2. **React** - Modern frontend web development
3. **Python PCEP** - Python fundamentals and certification prep
4. **Java OCP** - Java programming and Oracle certification
5. **AWS Serverless** - Cloud-native serverless architecture
6. **Lambda Python** - AWS Lambda function development
7. **DynamoDB Single Table** - NoSQL database design
8. **SQS** - Message queue architecture
9. **AgentCore** - AI agent framework development
10. **RDS MySQL** - Relational database management

---

## Learning Framework: 3-Level Progression

### Level 1: Beginner (Foundation)
**Duration**: 4-8 weeks per technology
**Focus**: Core concepts, syntax, basic operations
**Goal**: Build fundamental understanding and simple projects

### Level 2: Intermediate (Application)
**Duration**: 8-12 weeks per technology
**Focus**: Real-world applications, best practices, integration
**Goal**: Build production-ready applications

### Level 3: Advanced (Mastery)
**Duration**: 12-16 weeks per technology
**Focus**: Architecture, optimization, advanced patterns, certification
**Goal**: Expert-level proficiency and professional certification

### Knowledge Checks
- Weekly quizzes (10%)

### Grading Scale
- **A (90-100%)**: Advanced mastery

- **C (70-79%)**: Competent
- **D (60-69%)**: Needs improvement

- Minimum 80% overall score
- Complete all projects
- Pass final assessment
- Demonstrate proficiency in skills
