# Performance Analytics Skill

**Skill Type**: School Administration - Data Analytics
**Version**: 1.0.0
**Parent Agent**: Headmaster Assistant Agent

---

## Skill Overview

Comprehensive school performance analytics, data visualization, trend analysis, and actionable insights for data-driven decision making.

---

## Capabilities

### 1. Enrollment Analytics
- Total enrollment tracking
- Grade-level distribution
- Demographic breakdowns
- Enrollment trends (YoY)
- Capacity utilization

### 2. Attendance Analytics
- Average daily attendance (ADA)
- Chronic absenteeism rates
- Attendance patterns by grade
- Tardy trends
- Absence reasons analysis

### 3. Academic Performance
- School-wide GPA
- Grade distribution
- Honor roll percentage
- Students at risk (<2.0 GPA)
- Subject-specific performance

### 4. Standardized Testing
- Proficiency rates by subject
- Growth percentiles
- Achievement gaps
- Year-over-year comparisons
- Subgroup performance

## Analytics Workflows

### Monthly Performance Review
```
1. Generate performance dashboard
2. Analyze key metrics
3. Identify trends
4. Compare to benchmarks
5. Flag areas of concern
6. Create action plans
7. Present to leadership team
```

### Data-Driven Interventions
```
1. Identify struggling students (data)
2. Analyze patterns and root causes
3. Design targeted interventions
4. Implement support programs
5. Monitor progress metrics
6. Evaluate effectiveness
7. Adjust strategies as needed
```

## Best Practices

### Data Collection
- Ensure data accuracy
- Regular validation
- Consistent definitions
- Timely entry
- Complete records

### Analysis
- Use multiple data points
- Look for patterns
- Consider context
- Compare to benchmarks
- Disaggregate by subgroups

### Reporting
- Clear visualizations
- Executive summaries
- Actionable insights
- Regular updates
- Stakeholder-appropriate detail
