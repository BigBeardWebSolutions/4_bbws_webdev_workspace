# Progress Monitoring Skill

**Skill Type**: Parent Support - Academic Monitoring
**Version**: 1.0.0
**Parent Agent**: Parents Agent

---

## Skill Overview

Comprehensive student progress monitoring strategies for parents, including grade interpretation, trend analysis, early warning identification, and actionable support planning.

---

## Capabilities

### 1. Grade Monitoring
- Real-time grade access
- Current vs. previous performance
- Subject-by-subject analysis
- Missing assignment identification
- Grade trend analysis

### 2. Attendance Tracking
- Daily attendance status
- Absence and tardy patterns
- Chronic absenteeism risk
- Attendance impact on grades

### 3. Progress Reporting
- Weekly summaries
- Monthly snapshots
- Quarter/semester report cards
- Standardized test results

---

## Monitoring Workflows

### Daily Check (5 minutes)
```
1. Log into parent portal
2. Check for new grades posted
3. Review any teacher messages
4. Check attendance for today
5. Note any missing assignments
6. Take action if urgent
```

### Weekly Review (15-20 minutes)
```
1. Review all current grades
   ├── Check each subject
   ├── Compare to last week
   ├── Identify trends (improving/declining)
   └── Note any concerns

2. Check assignment completion
   ├── List missing assignments
   ├── Note upcoming due dates
   ├── Review submission status
   └── Plan catch-up if needed

3. Review attendance
   ├── Count absences this week
   ├── Check for patterns
   ├── Address attendance issues
   └── Submit excuses if needed

4. Read teacher feedback
   ├── Review recent comments
   ├── Note behavior observations
   ├── Identify positive notes
   └── Address concerns

5. Take action
   ├── Praise improvements
   ├── Address concerns with child
   ├── Contact teacher if needed
   └── Adjust home support plan
```

## Grade Interpretation Guide

### Understanding Grading Systems

#### Traditional Letter Grades
```
A (90-100%): Excellent work, exceeding standards
B (80-89%):  Good work, meeting standards well
C (70-79%):  Satisfactory, meeting basic standards
D (60-69%):  Below expectations, needs improvement
F (<60%):    Not meeting standards, significant concern
```

#### Standards-Based Grading
```
4 - Advanced:    Exceeds grade-level standards
3 - Proficient:  Meets grade-level standards
2 - Developing:  Approaching standards, needs support
1 - Beginning:   Below standards, significant support needed
```

### What Grades Tell You

**90-100% (A)**: Student is mastering content
- **Action**: Encourage and challenge
- **Support**: Provide enrichment opportunities

**80-89% (B)**: Student is doing well
- **Action**: Maintain current support
- **Support**: Help with occasional challenges

**70-79% (C)**: Student is meeting minimum expectations
- **Action**: Monitor closely, increase support
- **Support**: Daily homework review, study help

**60-69% (D)**: Student is struggling
- **Action**: Immediate intervention needed
- **Support**: Contact teacher, consider tutoring

**Below 60% (F)**: Student needs significant help
- **Action**: Emergency intervention
- **Support**: Teacher meeting, tutoring, possible evaluation

## Trend Analysis

### Identifying Patterns

#### Improving Trend
```
Week 1: 75%
Week 2: 78%
Week 3: 82%
Week 4: 85%

Signal: Positive! Student is mastering content.
Action: Praise effort, maintain current support.
```

#### Declining Trend
```
Week 1: 88%
Week 2: 85%
Week 3: 79%
Week 4: 73%

Signal: Warning! Student may be struggling.
Action: Investigate cause, increase support, contact teacher.
```

#### Inconsistent Performance
```
Week 1: 90%
Week 2: 65%
Week 3: 88%
Week 4: 70%

Signal: Uneven! May indicate test anxiety, understanding gaps.
Action: Discuss with student and teacher, identify specific issues.
```

## Early Warning Signs

### Academic Red Flags
- Sudden grade drop (10+ points)
- Multiple missing assignments
- Failing or near-failing grades
- Declining participation
- Test scores much lower than classwork

### Attendance Red Flags
- Frequent absences (>10% of days)
- Pattern of Monday/Friday absences
- Chronic tardiness
- Avoiding specific classes

### Behavior Red Flags (from teacher notes)
- "Not engaged in class"
- "Frequently off-task"
- "Not turning in work"
- "Seems overwhelmed"
- "Conflicts with peers"

## Action Planning

### When Grades Are Good (A-B)
```
✓ Acknowledge and praise
✓ Ask what's working
✓ Maintain current routine
✓ Provide enrichment if interested
✓ Encourage goal-setting
```

### When Grades Are Slipping (C)
```
1. Talk with your child
   - "I noticed your math grade dropped. What's happening?"
   - Listen without judgment
   - Identify obstacles

2. Increase support at home
   - Daily homework check
   - Dedicated study time
   - Remove distractions
   - Provide supplies/resources

3. Monitor weekly
   - Check portal more frequently
   - Track improvement
   - Adjust support as needed

4. Contact teacher if no improvement in 2 weeks
```

### When Grades Are Failing (D-F)
```
1. Immediate teacher contact
   - Email or call within 24 hours
   - Ask specific questions:
     * What content is student missing?
     * What can be done to improve?
     * Is tutoring available?
     * Can missing work be made up?

2. Create intervention plan
   - Daily homework supervision
   - Study sessions (30-60 min/day)
   - Tutoring if available
   - Teacher office hours
   - Limit distractions at home

3. Schedule face-to-face conference
   - Discuss root causes
   - Review intervention plan
   - Set concrete goals
   - Establish communication schedule

4. Consider additional support
   - Peer tutoring
   - After-school programs
   - Private tutoring
   - Educational evaluation (if pattern across subjects)
```

## Communication Scripts

### Email Template: Checking In
```
Subject: Checking in on [Student Name]'s progress in [Subject]

Dear [Teacher],

I've been monitoring [Student]'s grade in your class and noticed
[he/she] currently has a [grade]. I wanted to reach out to better
understand [his/her] performance.

Specifically, I'm wondering:
- Are there any areas where [he/she] is particularly struggling?
- What can we do at home to support [his/her] learning?
- Are there specific assignments or skills we should focus on?

I'm committed to supporting [Student]'s success in your class.
Please let me know if you have suggestions or if we should schedule
a time to talk.

Thank you,
[Your Name]
```

### Email Template: Missing Assignments
```
Subject: Help with Missing Assignments - [Student Name]

Dear [Teacher],

I noticed [Student] has [number] missing assignments in your class:
- [Assignment 1] - Due [Date]
- [Assignment 2] - Due [Date]

I want to help [him/her] get caught up. Can you please clarify:
- Can these assignments still be submitted?
- Is there a late penalty?
- What are the requirements for each?

I will work with [Student] this week to complete these. Please let
me know if you have any specific guidance.

Thank you,
[Your Name]
```

## Best Practices

### DO:
- Check portal at least 3x per week
- Respond promptly to teacher messages
- Praise improvements and effort
- Ask your child about their day/classes
- Create consistent homework routine
- Address concerns early
- Partner with teachers

### DON'T:
- Wait until report cards to check grades
- Punish without understanding root cause
- Do homework for your child
- Ignore teacher communications
- Compare children to each other or peers
- Overreact to single bad grade
- Blame teachers for low grades

## Troubleshooting

### "Portal shows different grade than child reports"
- Trust the portal (official record)
- Ask child to explain discrepancy
- Check for recent updates
- Clarify grading policy with teacher

### "Grade seems incorrect"
- Review grading breakdown
- Check for missing assignments counted as zeros
- Email teacher to clarify
- Request grade recalculation if error found

### "Child says assignment was turned in but shows missing"
- Have child show you submission confirmation
- Email teacher with details
- Ask child to resubmit if needed
- Teach importance of keeping proof of submission
